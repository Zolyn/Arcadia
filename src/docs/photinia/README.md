## 设计初衷
每当我新建一个项目的时候，都要手动把开发环境搭建并配置好，在这过程中会浪费大量的时间，所以我以“模板”这个概念为基础，开发了这样一个工具

## 功能
- 自动为项目生成`package.json`文件（如果没有的话）
- 询问式交互界面（基于 Inquirer.js）
- 自动复制模板文件（文件夹）到当前位置
- 可自行选择从模板中导入的文件、开发依赖和 npm 脚本
- 模板可扩展
- 开发中...
